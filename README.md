# Sounds & Words 🐹

Two gentle phonics games for early readers:

- **🔤 Letter Sounds** — hear a sound, see the keyword picture, tap the matching letter.
- **📖 Sight Words** — find the two words that are the same.

A cute capybara grows bigger each time you feed it a cupcake for a correct answer.

Built to be calm and low-pressure: no timers, no scores, nothing to lose.

## Play it
👉 **https://joy48poop.github.io/letter-sounds/**

Works in any web browser on a phone, tablet, or computer. Turn the sound on.

## Run it offline
Open `index.html` in a web browser, or serve the folder with any static web server.
